HTML 기초 태그

기초적인 HTML 태그를 공부해보자

시맨틱 태그

태그 기능
<header> 페이지나 특정 영역의 머리 부분을 나타내는 태그
<main> 페이지의 핵심 내용을 표시하는 태그
<footer> 페이지의 바닥글을 담당하는 태그

텍스트 태그

태그 기능
<hn> 제목을 표시하며, 숫자 1~6으로 크기 조절 가능
<p> 단락을 작성 가능한 태그
<br> 줄을 바꾸는 태그로, 닫는 태그가 없음
<b> bold, 글자를 두껍게 표시하는 태그
<i> italic, 텍스트를 기울임꼴로 표시하는 태그

테이블 관련 태그

태그 기능
<table> 테이블을 만드는 태그
<th> 테이블의 행, 열의 머리말(table heading)을 나타내는 태그
<td> 테이블의 내용(table data)을 표현하는 태그
